Screen Match : Phrase-Generator

Screen-Match : Phrase-Generator es una aplicación FullStack que permite a los usuarios visualizar los datos proporcionados por películas y series. Esta aplicación está construida con un backend robusto utilizando Java, Spring Boot, JPA, y PostgreSQL, mientras que el frontend está desarrollado con HTML, CSS, y JavaScript para una experiencia de usuario interactiva.
